GRACE-X BOOT VIDEO DIRECTORY
=============================

Drop your boot video here as:
  - gracex-boot.mp4 (recommended)
  - Any .mp4, .webm, or .mov file

Recommended specs:
  - Resolution: 1920x1080 or 3840x2160
  - Duration: 3-6 seconds
  - Format: MP4 (H.264)
  - Size: Under 10MB for fast loading

To integrate your video into the boot screen:
  1. Place video file here
  2. Edit assets/js/boot-screen.js
  3. Find createBootScreen() method
  4. Add video element (see BOOT_SCREEN_README.md for code)

Your current boot screen uses the static logo image.
Add a video for even more impact!
